# 🏥 Hospital Operations & Supply Chain Command Centre

An interactive **Power BI dashboard** built to monitor hospital performance and supply chain operations through real-time KPIs and insightful visualizations.
# Project Overview
This dashboard integrates patient, staff, inventory, financial, and vendor data into a single reporting solution. Using Power Query for data preparation and DAX for analytical calculations, the dashboard provides real-time insights into hospital performance, patient trends, inventory management, and operational efficiency.

The objective of this project is to demonstrate practical business intelligence skills by designing a professional dashboard that supports data-driven healthcare management.

## 🚀 Highlights

-  Executive Dashboard Design
-  Interactive KPIs & Slicers
-  Patient Admission & Diagnosis Analysis
-  Inventory & Stock Monitoring
-  Staff Overtime Analysis
-  DAX Measures & Power Query Transformations
-  Professional Healthcare Analytics Dashboard

## 📌 Dashboard Features

- Total Patients
- Average Length of Stay
- Total Overtime Hours
- Current Stock
- Low Stock Items
- Daily Patient Admissions Trend
- Top Diagnoses
- Inventory Distribution
- Room Type Analysis
- Inventory Stock Status

## 🛠 Tech Stack

- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV
